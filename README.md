# Sleep & Work Productivity Analysis

A survey-based research project analyzing the relationship between sleep patterns 
and work productivity across 127 respondents.

## What the project does
Collects and analyzes survey data on sleep habits and work productivity. 
Builds 6 composite indices from 20 Likert-scale items, then applies 
descriptive and inferential statistical methods to uncover how sleep 
quality impacts cognitive performance, emotional wellbeing, and overall productivity.

## Why the project is useful
Poor sleep is one of the most overlooked factors affecting workplace performance. 
This project quantifies that relationship using real survey data and rigorous 
statistical analysis, making it useful for researchers, HR professionals, 
and anyone interested in data-driven wellbeing insights.

## How to get started
1. Clone or download the repository
2. Install dependencies:
   ```
   pip install pandas numpy matplotlib seaborn scipy statsmodels openpyxl
   ```
3. Run the notebooks in order (1 → 5) in Jupyter or any compatible environment

## Project Structure
1. `1_Data_Cleaning.ipynb` — data cleaning and preprocessing
2. `2_Index_Prep.ipynb` — construction of 6 composite indices
3. `3_univariate_desc_analysis.ipynb` — univariate EDA
4. `4_bivariate_desc_analysis.ipynb` — bivariate EDA and visualizations
5. `5_inf_analysis.ipynb` — t-test, ANOVA, and Pearson correlation analysis

## Key Numbers
- 127 survey respondents
- 6 composite indices constructed
- 4 inferential tests applied
